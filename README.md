# JQAS(JIIT Query Addressal System) App
Having an experience of online classes and being so called the ‘Online Batch’ during the Covid – 19 time, we experienced the lack of conversation with the students already studying in the college seeking information of the college, thus we tried to ease the mode of conversation between the students of the college by trying to develop an Android Application named as “JQAS – JIIT Query Addressal System ”, an application that provides a common platform for the students for the interaction with other students and to remain updated about the college. This application is for the college student for them to chat with seniors, juniors and students within the same year. With the rise of the use of the android smartphones and the reach of each person to access the smartphone, we chose the android platform to develop the application.
